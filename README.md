## 🎉 Getting Started

To get a local copy up and running, follow these simple steps:

1. **Clone the repo**:
    ```sh
    git clone https://github.com/rafael-a-g-n/Desmoto-Arts-Project.git
    ```
2. **Open the index.html file in your browser** to see the magic! ✨
